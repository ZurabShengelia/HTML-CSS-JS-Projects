🚀 Vanilla Web Projects Collection
A curated collection of modern, high-performance web applications built exclusively with vanilla HTML5, CSS3, and JavaScript (ES6+). This repository showcases complex state management, custom UI systems, and interactive environments without the use of external frameworks or libraries.

🛠️ Featured Projects
1. 📄 ResuméCraft — Professional Resume Builder
A feature-rich SPA for creating professional resumes with real-time preview.

Key Tech: Vanilla JS State Management, LocalStorage persistence, CSS Grid/Flexbox.

Highlights: 6 templates, 5 color schemes, and auto-draft saving.

2. 🖥️ SamtrediaOS — Browser-Based OS
A fully functional browser-based operating system with a windowing system and built-in apps.

Key Tech: Custom Filesystem logic, Terminal emulator, Glassmorphism UI.

Highlights: Integrated Local AI assistant, Terminal with 20+ commands, and a multi-note editor.

3. 🪐 Solar System Explorer
An interactive 3D solar system visualization using WebGL.

Key Tech: Three.js, Procedural Shaders, J2000 ephemeris data.

Highlights: Realistic orbital mechanics, atmospheric rim-lighting, and asteroid belt simulation.

4. 🏋️ GEO Supplements — E-Commerce Store
A modern e-commerce platform for fitness supplements.

Key Tech: IntersectionObserver for animations, dynamic product filtering, shopping cart logic.

Highlights: Particle background, mobile-responsive grid, and automated checkout calculation.

5. ⌨️ TypeFast — Typing Speed Test
A brutalist-editorial typing test focused on precision and performance.

Key Tech: Web Audio API (pitch-varied sounds), Canvas API for WPM charts.

Highlights: Heatmap of keyboard mistakes, CMYK misprint effects, and local leaderboards.

🎨 Design Philosophy
Every project in this repository follows a specific aesthetic and technical standard:

Zero Dependencies: No React, Vue, or npm packages. Pure web technologies only.

Minimalist & Dark Aesthetics: High-contrast UIs, gold accents, and fluid animations.

Performance First: Optimized for fast load times and smooth 60fps interactions.

Responsive: Full compatibility across desktop, tablet, and mobile devices.
